##  Overview

The **Smart Irrigation System** is an AI-driven platform designed to optimize modern farming by integrating:

-  Environmental data  
-  Machine learning models  
-  Real-time monitoring systems  

It helps farmers **increase yield**, **reduce water wastage**, and **automate decision-making**.

---

##  Features

| Feature | Description |
|---|---|
|  **Crop Recommendations** | ML-based suggestions using soil & climate data |
|  **Cattle Monitoring** | Detects unusual movement & triggers alerts |
|  **Smart Irrigation** | Automates watering based on live sensor data |
|  **Interactive Dashboard** | Clean and responsive UI |
|  **Real-time Sync** | Powered by Supabase backend |
